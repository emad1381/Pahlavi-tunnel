# 📌 معرفی پروژه

emad یک سیستم تونل TCP معکوس است که برای اتصال دو سرور طراحی شده است:

- 🇮🇷 سرور ایران (IR)
- 🌍 سرور خارج (EU)

این سیستم از چندین قابلیت حرفه‌ای مانند مدیریت چند اسلات، همگام‌سازی خودکار پورت‌ها، بررسی سلامت سرویس و بهینه‌سازی شبکه پشتیبانی می‌کند.

---

# 🧠 معماری سیستم

```
کاربر → سرور IR ⇄ سرور EU
                │
        Bridge Port (اتصال اصلی)
                │
         Sync Port (همگام‌سازی)
```

### 🔹 Bridge Port
پورت اصلی ارتباط پایدار بین سرور ایران و خارج است.  
تمام ترافیک TCP از این پورت عبور می‌کند.

### 🔹 Sync Port
برای هماهنگ‌سازی خودکار پورت‌ها بین دو سرور استفاده می‌شود.

---

# 🛠 قابلیت‌ها

| قابلیت | توضیح |
|---------|--------|
| Reverse TCP Tunnel | اتصال پایدار بین IR و EU |
| Multi-Slot (1–10) | ذخیره ۱۰ کانفیگ مختلف |
| AutoSync | ایجاد و حذف خودکار پورت‌ها |
| Cron Health Check | ریستارت خودکار در صورت توقف |
| بهینه‌سازی BBR | افزایش کارایی شبکه |
| چند روش Port Forward | iptables / nftables / HAProxy / socat |
| systemd Integration | اجرای خودکار بعد از ریبوت |
| کنترل Thread | جلوگیری از مصرف بیش از حد منابع |
| تنظیمات پیشرفته | کنترل از طریق ENV |

# 🟢 مرحله اول — نصب روی سرور ایران

```bash
bash <(curl -Ls https://raw.githubusercontent.com/emad1381/emad/main/install.sh)
```

برای اجرای منوی مدیریت تانل دستور زیر را وارد کنید:

```bash
sudo emad
```


---

## 1️⃣ نصب کامل

از منو انتخاب کنید:

```
5) Install / Complete Setup
```

این گزینه:
- وابستگی‌ها را نصب می‌کند
- سرویس systemd ایجاد می‌کند
- ساختار موردنیاز را آماده می‌کند

---

## 2️⃣ ایجاد تانل

```
1) Create Tunnel
2) IRAN Server
```

---

## 3️⃣ انتخاب Slot (1 تا 10)

هر Slot یک کانفیگ جداگانه ذخیره می‌کند.

---

## 4️⃣ وارد کردن Bridge Port

پیش‌فرض:

```
7000
```

باید در هر دو سرور یکسان باشد.

---

## 5️⃣ وارد کردن Sync Port

پیش‌فرض:

```
7001
```

باید در هر دو سرور یکسان باشد.

---

## 6️⃣ فعال‌سازی AutoSync

اگر می‌خواهید همگام‌سازی خودکار فعال باشد:

```
y
```

در غیر این صورت:

```
n
```

---

## 7️⃣ وارد کردن پورت کانفیگ

پورت سرویس مورد نظر خود را وارد کنید.

در پایان:

```
Enter
```

# ❓ سوالات متداول

### آیا Bridge و Sync باید یکسان باشند؟
بله، در هر دو سرور باید دقیقاً یکسان باشند.

### آیا می‌توان چند تانل داشت؟
بله، از Slotهای مختلف استفاده کنید.

### آیا بعد از ریبوت اجرا می‌شود؟
بله، از طریق systemd.
